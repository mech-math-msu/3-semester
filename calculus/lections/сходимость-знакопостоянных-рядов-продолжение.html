Сходимость знакопостоянных рядов: продолжение

Схема Куммера

Теорема: (схема Куммера или фабрика по производству новых признаков сходимости)

Рассмотрим ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) и последовательность \(\{c_n\}_{n = 1}^{\infty}\). Пусть \(a_n, c_n > 0 \,\,\,\, \forall n \in \mathbb{N}\).

  1. Если \(\gamma_n = c_n \cdot \frac{a_n}{a_{n + 1}} - c_{n + 1} \geqslant \alpha > 0\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится.

  2. Если ряд \(\displaystyle \sum_{n = 1}^{\infty}\frac1{c_n}\) расходится и \(\gamma_n = c_n \cdot \frac{a_n}{a_{n + 1}} - c_{n + 1} \leqslant 0\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) расходится.

Доказательство:

  1. \[c_n \cdot \frac{a_n}{a_{n + 1}} - c_{n + 1} \geqslant \alpha > 0\] \[c_n \cdot a_n - c_{n + 1}\cdot a_{n + 1}\geqslant \alpha\cdot a_{n + 1} > 0\]

Суммируем от \(1\) до \(N\):

\[c_1 \cdot a_1 - c_{N + 1}\cdot a_{N + 1} \geqslant \alpha \cdot \displaystyle \sum_{n = 2}^{N}a_n\]

Частичные суммы ограничены, и члены ряда положительны, а значит по признаку ряд сходится.

  1. \[c_n \cdot \frac{a_n}{a_{n + 1}} - c_{n + 1} \leqslant 0\]

\[\frac{a_n}{a_{n + 1}} \leqslant \frac{\frac1{c_n}}{\frac1{c_{n + 1}}}\]

Перемножаем от \(1\) до \(N\):

Все начинает со страшной силой сокращаться…

\[\frac{a_1}{a_{N + 1}} \leqslant \frac{\frac1{c_1}}{\frac1{c_{N + 1}}}\] \[a_{N + 1} \geqslant \frac{a_1c_1}{c_{N + 1}}\]

По признаку сравнения ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) расходится \(\,\,\,\,\blacksquare\)

Следствя из схемы Куммера: признаки Раабе, Бертрана и Гаусса

Теорема: (признак Раабе) Рассмотрим ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\). Пусть \(a_n > 0 \,\,\,\, \forall n \in \mathbb{N}\).

Тогда, если \(n\cdot\left(\frac{a_n}{a_{n + 1}} - 1\right) > 1\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится.

Если \(n\cdot\left(\frac{a_n}{a_{n + 1}} - 1\right) \leqslant 1\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) расходится.

Доказательство: \(c_n = n \,\,\,\,\blacksquare\)

Почему вообще кто-то придумал признак Раабе? Потому что в признаке Д’Аламбера в \(1\) нельзя ничего сказать про сходимость, поэтому вычитаем \(1\) и смотрим, что можно сделать. 1

Следствие: (признак Раабе в предельной форме) Рассмотрим ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\). Пусть \(a_n > 0 \,\,\,\, \forall n \in \mathbb{N}\).

Тогда, если \(\exists \displaystyle \lim_{n\to \infty}\left(n\cdot\left(\frac{a_n}{a_{n + 1}} - 1\right)\right) > 1\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится.

Если \(\exists \displaystyle \lim_{n\to \infty}\left(n\cdot\left(\frac{a_n}{a_{n + 1}} - 1\right)\right) < 1\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) расходится.

Доказательство: Понять почему исчезает равенство \(\,\,\,\,\blacksquare\)

Теорема: (Признак Бертрана) Рассмотрим ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\). Пусть \(a_n > 0 \,\,\,\, \forall n \in \mathbb{N}\). Тогда

  1. Если \(\ln(n)\left[ n\left(\frac{a_n}{a_{n + 1}} - 1\right) - 1\right] > 1\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится.
  2. Если \(\ln(n)\left[ n\left(\frac{a_n}{a_{n + 1}} - 1\right) - 1\right] \leqslant 1\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) расходится.

Доказательство: \(c_n = n \ln(n)\)

\[\gamma_n = n \ln(n) \frac{a_n}{a_{n + 1}} - (n + 1) \ln(n + 1)=\] \[= n \ln(n) \frac{a_n}{a_{n + 1}} + n \ln(n) - n \ln(n) - (n + 1) \ln(n + 1)=\] \[= n \ln(n) \left(\frac{a_n}{a_{n + 1}} - 1\right) + n \ln(n) - \ln(n) + \ln(n) - (n + 1) \ln(n + 1)=\] \[= \ln(n)\left[ n\left(\frac{a_n}{a_{n + 1}} - 1\right) - 1\right] + (n + 1) \ln(n) - (n + 1) \ln(n + 1)=\] \[= \ln(n)\left[ n\left(\frac{a_n}{a_{n + 1}} - 1\right) - 1\right] + (n + 1) \ln\left(1 - \frac1{n + 1}\right)=\] \[= \ln(n)\left[ n\left(\frac{a_n}{a_{n + 1}} - 1\right) - 1\right] + (n + 1)\left(-\frac1{n + 1} + \overline{\overline{o}}\left(-\frac1n\right)\right)=\] \[= \ln(n)\left[ n\left(\frac{a_n}{a_{n + 1}} - 1\right) - 1\right] -\frac{n}{n + 1} -\frac1{n + 1} + \overline{\overline{o}}\left(-\frac1n\right)=\] \[= \ln(n)\left[ n\left(\frac{a_n}{a_{n + 1}} - 1\right) - 1\right] -1 + \underline{\underline{O}}\left(\frac1n\right)=\] \[= \ln(n)\left[ n\left(\frac{a_n}{a_{n + 1}} - 1\right) - 1\right] -1 + \beta_n =\]

\(\beta_n = \underline{\underline{O}}\left(\frac1n\right)\) и \(\beta_n < 0 \,\,\,\, \forall n \in \mathbb{N}\), а значит по схеме Куммера:

  1. Если \(\ln(n)\left[ n\left(\frac{a_n}{a_{n + 1}} - 1\right) - 1\right] > 1\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится.
  2. Если \(\ln(n)\left[ n\left(\frac{a_n}{a_{n + 1}} - 1\right) - 1\right] \leqslant 1\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) расходится \(\,\,\,\,\blacksquare\)

Следствие: (признак Бертрана в предельной форме) Рассмотрим ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\). Пусть \(a_n > 0 \,\,\,\, \forall n \in \mathbb{N}\). Тогда

  1. Если \(\exists\displaystyle \lim_{n\to \infty}\ln(n)\left[ n\left(\frac{a_n}{a_{n + 1}} - 1\right) - 1\right] > 1\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится.
  2. Если \(\exists\displaystyle \lim_{n\to \infty}\ln(n)\left[ n\left(\frac{a_n}{a_{n + 1}} - 1\right) - 1\right] < 1\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) расходится.

Доказательство: \(\,\,\,\,\blacksquare\)

Теорема: (признак Гаусса) Рассмотрим ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\). Пусть \(a_n > 0 \,\,\,\, \forall n \in \mathbb{N}\) и \(\frac{a_n}{a_{n + 1}} = \lambda + \frac{\mu}{n} + \underline{\underline{O}}\left(\frac{1}{n^{1 + \delta}}\right), \,\,\,\, \delta > 0\). Тогда

  • Если \(\lambda > 1\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится,
  • Если \(\lambda < 1\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) расходится,
  • Если \(\lambda = 1\), то – если \(\mu > 1\) ряд сходится, – если \(\mu \leqslant 1\) ряд расходится.

Доказательство:

Если \(\lambda > 1\) или \(\lambda < 1\), то по признаку Д’Аламбера в предельной форме.

Если \(\lambda = 1\) и \(\mu > 1\) или \(\mu < 1\), то по признаку Раабе. 2

Если \(\lambda = 1\) и \(\mu = 1\), то \(\frac{a_n}{a_{n + 1}} = 1 + \frac1{n} + \underline{\underline{O}}\left(\frac{1}{n^{1 + \delta}}\right)\). Домножим на \(n\ln(n)\):

\[\ln(n)\left[n\left(\frac{a_n}{a_{n + 1}} - 1\right) - 1\right] = \underline{\underline{O}}\left(\frac{\ln(n)}{n^{\delta}}\right) \implies\]

\[\displaystyle \lim_{n\to \infty}\ln(n)\left[n\left(\frac{a_n}{a_{n + 1}} - 1\right) - 1\right] = 0\]

Значит по признаку Бертрана в предельной форме ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) расходится \(\,\,\,\,\blacksquare\)

Абсолютная сходимость

Определение: Ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится абсолютно, если сходится ряд \(\displaystyle \sum_{n = 1}^{\infty}|a_n|\).

Если ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится, а ряд \(\displaystyle \sum_{n = 1}^{\infty}|a_n|\) расходится, то \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится условно.

Теорема: Если ряд \(\displaystyle \sum_{n = 1}^{\infty}|a_n|\) сходится, то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится.

Доказательство: Критерий Коши и \(\displaystyle \left|\sum_{n = k + 1}^{m}a_n\right| \leqslant \left|\sum_{n = k + 1}^{m}|a_n|\right| < \varepsilon \,\,\,\,\blacksquare\)

Определение: Биекция \(\sigma: \mathbb{N} \to \mathbb{N}\) – перестановка.

Лемма: Пусть \(\sigma: \mathbb{N} \to \mathbb{N}\) – биекция, ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится и \(a_n \geqslant 0 \,\,\,\, \forall n \in \mathbb{N}\). Тогда ряд \(\displaystyle \sum_{n = 1}^{\infty}a_{\sigma(n)}\) сходится.

Доказательство: Обозначим частичные суммы рядов \(\displaystyle \sum_{n = 1}^{\infty}a_n\) и \(\displaystyle \sum_{n = 1}^{\infty}a_{\sigma(n)} \,\,\,\, S_k\) и \(S^{\sigma}_k\) соответственно. Пусть \(N = \displaystyle\max_{0 \leqslant n \leqslant k}\{\sigma(n)\}\). Тогда \(S^{\sigma}_k \leqslant S_N\,\,\,\,\blacksquare\)

Теорема: Пусть \(\sigma: \mathbb{N} \to \mathbb{N}\) – биекция, ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится абсолютно.

Тогда ряд \(\displaystyle \sum_{n = 1}^{\infty}a_{\sigma(n)}\) сходится абсолютно и \(\displaystyle \sum_{n = 1}^{\infty}a_n = \displaystyle \sum_{n = 1}^{\infty}a_{\sigma(n)}\).

Доказательство:

Рассмотрим ряды \(\displaystyle \sum_{n = 1}^{\infty}(a_n + |a_n|)\) и \(\displaystyle \sum_{n = 1}^{\infty}(a_{\sigma(n)} + |a_{\sigma(n)}|)\).

Это ряды с неотрицательными членами и ряд \(\displaystyle \sum_{n = 1}^{\infty}(a_n + |a_n|)\) сходится, как сумма двух сходящихся рядов, а значит по лемме ряд \(\displaystyle \sum_{n = 1}^{\infty}(a_{\sigma(n)} + |a_{\sigma(n)}|)\) сходится. Следовательно, ряд \(\displaystyle \sum_{n = 1}^{\infty}|a_{\sigma(n)}|\) сходится, как разность двух сходящихся рядов \(\,\,\,\,\blacksquare\)

Определение: Рассмотрим ряды \(\displaystyle \sum_{n = 1}^{\infty}a_n\) и \(\displaystyle \sum_{k = 1}^{\infty}b_k\). Пусть \(\varkappa: \mathbb{N} \to \mathbb{N} \times \mathbb{N}\) – биекция. \(\displaystyle \sum_{\substack{\varkappa = 1\\ \varkappa = \varkappa(n, k)}}^{\infty}a_n\cdot b_k\) – произведение рядов \(\displaystyle \sum_{n = 1}^{\infty}a_n\) и \(\displaystyle \sum_{k = 1}^{\infty}b_k\) по правилу \(\varkappa\).

Теорема: Пусть ряды \(\displaystyle \sum_{n = 1}^{\infty}a_n\) и \(\displaystyle \sum_{k = 1}^{\infty}b_k\) сходятся абсолютно, \(\varkappa: \mathbb{N} \to \mathbb{N} \times \mathbb{N}\) – биекция. Тогда ряд \(\displaystyle \sum_{\substack{\varkappa = 1\\ \varkappa = \varkappa(n, k)}}^{\infty}a_n\cdot b_k\) сходится и \(\displaystyle \sum_{\substack{\varkappa = 1\\ \varkappa = \varkappa(n, k)}}^{\infty}a_n\cdot b_k = A \cdot B\), где \(A = \displaystyle \sum_{n = 1}^{\infty}a_n\) и \(B = \displaystyle \sum_{k = 1}^{\infty}b_k\).

Доказательство:

Записываем члены ряда в табличку:

\[\begin{array}{cccc} a_1b_1 & a_2b_1 & a_3b_1 & \ldots\\ a_1b_2 & a_2b_2 & a_3b_2 & \dots\\ a_1b_3 & a_2b_3 & a_3b_3 & \dots\\ a_1b_4 & a_2b_4 & a_3b_4 & \dots\\ \vdots & \vdots & \vdots & \ddots\\\end{array}\]

Нумеруем по уголкам.

Просто прикольная картинка

\[\displaystyle \sum_{\varkappa = 1}^{N^2}a_n b_k = \left(\displaystyle \sum_{n = 1}^{N}a_n\right)\cdot \left(\displaystyle \sum_{k = 1}^{N}b_k\right)\]

\[\left|\displaystyle \sum_{\varkappa = 1}^{N^2 + m}a_n b_k - \displaystyle \sum_{\varkappa = 1}^{N^2}a_n b_k\right| \leqslant |a_{N + 1}| \displaystyle \sum_{i = 1}^{N + 1}|b_i| + |b_{N + 1}| \displaystyle \sum_{j = 1}^{N}|a_j| \,\,\,\,\blacksquare\]
