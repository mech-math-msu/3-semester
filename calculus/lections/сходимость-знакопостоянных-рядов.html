Сходимость знакопостоянных рядов

Сначала надо обязательно проверять необходимое условие сходимости, а потом применять тяжелую артиллерию.

Знакопостоянные ряды

Замечание: Везде, где написано \(\forall n \in \mathbb{N}\), можно писать \(\exists k \in \mathbb{N}: \,\,\,\, \forall n > k\).

Определение: Если \(\forall n \in \mathbb{N} \,\,\,\, a_n \geqslant 0\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) – знакопостоянный.

Теорема: (сходимость знакопостоянного монотонного ряда) Пусть \(a_n \geqslant 0 \,\,\,\, \forall n \in \mathbb{N}\) и \(\{S_n\}_{n = 1}^{\infty}\) ограничена. Тогда ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится.

Доказательство: \(a_n \geqslant 0 \implies S_n\) монотонно возрастает \(\implies\) (\(\{S_n\}_{n = 1}^{\infty}\) ограничена) \(\exists \displaystyle \lim_{x\to \infty}S_n \,\,\,\,\blacksquare\)

Замечание: Ряд с неотрицательными членами сходится \(\iff\) сходится хотя бы одна подпоследовательность последовательности его частичных сумм.

Теорема: (признак сравнения) Пусть \(a_n \geqslant b_n \geqslant 0 \,\,\,\, \forall n \in \mathbb{N}\).

Тогда, если \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится, то \(\displaystyle \sum_{n = 1}^{\infty}b_n\) сходится.

Если \(\displaystyle \sum_{n = 1}^{\infty}b_n\) расходится, то \(\displaystyle \sum_{n = 1}^{\infty}a_n\) расходится.

Доказательство: \(0 \leqslant \displaystyle \sum^{N}b_n \leqslant \displaystyle \sum^{N}a_n \,\,\,\,\blacksquare\)

Теорема: (признак сравнения в предельной форме) Пусть \(a_n \geqslant b_n > 0 \,\,\,\, \forall n \in \mathbb{N}\). Пусть \(\exists \displaystyle \lim_{n\to \infty}\frac{a_n}{b_n} = R > 0\) 1. Тогда ряды \(\displaystyle \sum_{n = 1}^{\infty}a_n\) и \(\displaystyle \sum_{n = 1}^{\infty}b_n\) сходятся или расходятся одновременно.

Доказательство: \(\forall n \in \mathbb{N} \,\,\,\, \frac{R}{2} \leqslant \frac{a_n}{b_n} \leqslant \frac{3R}{2} \implies \frac12 R \cdot b_n \leqslant a_n \leqslant \frac32 R \cdot b_n \,\,\,\,\blacksquare\)

Интегральный признак Коши

Теорема: (интегральный признак Коши) Пусть \(f(x) \geqslant 0\) на \([0, +\infty)\) и \(f(x)\) монотонно убывает на \([1, +\infty)\). Пусть \(a_n = f(n)\). Тогда \(\displaystyle \sum_{n = 1}^{\infty}a_n\) и \(\displaystyle\int\limits_{1}^{+\infty} f(x)dx\) 2 сходятся или расходятся одновременно.

Доказательство: \[\forall x \in [n, n + 1] \,\,\,\, f(n + 1) \leqslant f(x) \leqslant f(n)\] Интегрируем от \(n\) до \(n + 1\): \[f(n + 1) \leqslant \displaystyle\int\limits_n^{n + 1} f(x)dx \leqslant f(n)\] Суммируем от \(1\) до \(N\): \[\displaystyle \sum_{n = 1}^{N}f(n + 1) \leqslant \displaystyle\int\limits_1^{N + 1} f(x)dx \leqslant \displaystyle \sum_{n = 1}^{N + 1}f(n)\] Далее по признаку сравнения \(\,\,\,\,\blacksquare\)

Пример: степенно-логарифмическая шкала

Признаки сходимости

Теорема: (признак Д’Аламбера) Пусть \(a_n > 0 \,\,\,\, \forall n \in \mathbb{N}\).

Если \(\forall n \in \mathbb{N} \,\,\,\, \frac{a_{n + 1}}{a_n} \geqslant 1\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) расходится.

Если \(\forall n \in \mathbb{N} \,\,\,\, \frac{a_{n + 1}}{a_n} \leqslant q < 1\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится.

Доказательство:

\(\forall n \in \mathbb{N} \,\,\,\, \frac{a_{n + 1}}{a_n} \geqslant 1 \implies a_{n + 1} \geqslant a_n \,\,\,\,\blacksquare\)

\(\forall n \in \mathbb{N} \,\,\,\, \frac{a_{n + 1}}{a_n} \leqslant q < 1 \implies a_{n + 1} \leqslant q \cdot a_n \implies a_{n + k} \leqslant q^k \cdot a_n \,\,\,\,\blacksquare\)

Теорема: (признак Д’Аламбера в предельной форме) Пусть \(a_n > 0 \,\,\,\, \forall n \in \mathbb{N}\).

\(\exists \displaystyle \lim_{n\to \infty}\frac{a_{n + 1}}{a_n} = \alpha\)

Если \(\alpha > 1\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) расходится.

Если \(\alpha < 1\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится.

Доказательство: \(\forall \varepsilon > 0 \,\,\,\, \exists N \in \mathbb{N}: \,\,\,\, \forall n > N \,\,\,\, \left|\frac{a_{n + 1}}{a_n} - \alpha \right| < \varepsilon \,\,\,\,\blacksquare\)

Столько всего названо в честь Коши, что приходится выкручиваться и это просто признак :)

Теорема: (признак Коши) Пусть \(a_n \geqslant 0 \,\,\,\, \forall n \in \mathbb{N}\).

Если \(\forall n \in \mathbb{N} \,\,\,\, \sqrt[n]{a_n} \geqslant 1\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) расходится.

Если \(\forall n \in \mathbb{N} \,\,\,\, \sqrt[n]{a_n} \leqslant q < 1\), то ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится.

Доказательство: \(\forall n \in \mathbb{N} \,\,\,\, \sqrt[n]{a_n} \leqslant q < 1 \implies a_n \leqslant q^n \,\,\,\,\blacksquare\)

Если \(\forall n \in \mathbb{N} \,\,\,\, \sqrt[n]{a_n} \geqslant 1\), то не выполнено необходимое условие сходимости \(\,\,\,\,\blacksquare\)
