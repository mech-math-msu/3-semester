Функциональные последовательности

Определения

Определение: Последовательность \(\{f_n: \,\,\,\, X \to \mathbb{R}\}_{n = 1}^{\infty}\) 1 – функциональная последовательность. 2

Определение: Функциональная последовательность \(\{f_n: \,\,\,\, X \to \mathbb{R}\}_{n = 1}^{\infty}\) ограничена на множестве \(X\), если \(\exists C > 0: \,\,\,\, \forall n \in \mathbb{N} \,\,\,\, \forall x \in X \,\,\,\, |f_n(x)| \leqslant C\).

Определение: Функциональная последовательность \(\{f_n: \,\,\,\, X \to \mathbb{R}\}_{n = 1}^{\infty}\) сходится (на множестве) \(X\), если \(\forall x_0 \in X\) последовательность \(\{f_n(x_0)\}_{n = 1}^{\infty}\) сходится.

Если функциональная последовательность сходится, то функция \(f: X \to \mathbb{R}, \,\,\,\, f: x_0 \mapsto \displaystyle \lim_{n\to \infty}f_n(x_0)\) – предел (функциональной) последовательности.

Обозначается: \(f_n \xrightarrow[X]{} f\)

В кванторах: \(\forall x_0 \in X \,\,\,\, \forall \varepsilon > 0 \,\,\,\, \exists N_{x_0} \in \mathbb{N}: \,\,\,\, \forall n > N_{x_0} \,\,\,\, |f_n(x_0) - f(x)| < \varepsilon\)

Определение: Функциональная последовательность \(\{f_n: \,\,\,\, X \to \mathbb{R}\}_{n = 1}^{\infty}\) равномерно сходится к функции \(f: X \to \mathbb{R}\), если \(\forall \varepsilon > 0 \,\,\,\, \exists N \in \mathbb{N}: \,\,\,\, \forall n > N \,\,\,\, \forall x \in X \,\,\,\, |f_n(x) - f(x)| < \varepsilon\).

Обозначается: \(f_n \underset{X}{\rightrightarrows} f\)

Критерии равномерной сходимости функциональной последовательности

Лемма: (критерий равномерной сходимости функциональной последовательности) \(f_n \underset{X}{\rightrightarrows} f \iff \displaystyle \sup_{X}|f_n(x) - f(x)| \underset{n \to \infty}{\rightarrow} 0\) 3

Доказательство: определение \(\,\,\,\,\blacksquare\)

Теорема: (критерий Коши равномерной сходимости функциональной последовательности) \(f_n \underset{X}{\rightrightarrows} f \iff \forall \varepsilon > 0 \,\,\,\, \exists N \in \mathbb{N}: \,\,\,\,\forall n, m > N \,\,\,\, \forall x \in X \,\,\,\, |f_n(x) - f_m(x)| < \varepsilon\).

Доказательство: \[\Longrightarrow\] Как обычно прибавить и вычесть \(f(x) \,\,\,\,\blacksquare\) \[\Longleftarrow\] Чего-то замутить 4 В каждой точке \(X\) последовательность функций сходится по критерию Коши, дальше взять \(n = m + p\) и устремить \(p\) к бесконечности \(\,\,\,\,\blacksquare\)
