Теорема Таубера и Литтлвуда

Теорема: (теорема Таубера) Пусть \(S(x) = \displaystyle \sum_{n = 0}^{\infty}a_nx^n, \,\,\,\, x \in [0, 1)\) и \(\exists \displaystyle \lim_{x\to 1-0}S(x) = A\). Если \(a_n = \overline{\overline{o}}\left(\frac1n\right)\) при \(n \to \infty\), то \(\displaystyle \sum_{n = 0}^{\infty}a_n = A\).

Доказательство:

Пусть \(N_x = \left[\frac{1}{1 - x}\right]\).

И \(\varphi(x) = \displaystyle \sum^{N_x}_{n = 0}a_n - \displaystyle \sum^{\infty}_{n = 0}a_n x^n\).

Доказать теорему значит доказать, что \(\displaystyle \lim_{x\to 1-0}\varphi(x) = 0\) 1.

\[\varphi(x) = \displaystyle \sum^{N_x}_{n = 0}a_n - \displaystyle \sum^{\infty}_{n = 0}a_n x^n = \displaystyle \sum^{N_x}_{n = 0}a_n(1 - x^n) - \displaystyle \sum^{\infty}_{n = N_x + 1}a_n x^n\]

\[\left|\displaystyle \sum^{\infty}_{n = N_x + 1}a_n x^n\right| \leqslant \displaystyle \sum^{\infty}_{n = N_x + 1}|a_n| x^n \leqslant \displaystyle \sum^{\infty}_{n = N_x + 1}\frac{\varepsilon}n x^n \leqslant\] \[\leqslant \frac{\varepsilon}{N_x} \displaystyle \sum^{\infty}_{n = N_x + 1}x^n \leqslant \frac{\varepsilon}{N_x} \displaystyle \sum^{\infty}_{n = 0}x^n = \frac{\varepsilon}{N_x} N_x = \varepsilon\]

\[\left|\displaystyle \sum^{N_x}_{n = 0}a_n(1 - x^n)\right| = \displaystyle \sum^{N_x}_{n = 0}|a_n|(1 - x)(1 + x^2 + \ldots + x^{n - 1})\leqslant\] \[\leqslant (1 - x)\displaystyle \sum^{N_x}_{n = 0}|a_n|n \leqslant \frac1{N_x} \displaystyle \sum^{N_x}_{n = 0}|a_n| \to 0, \,\,\,\, x \to 1-0\]

\[\Downarrow\]

\[|\varphi(x)|\leqslant \left|\displaystyle \sum^{N_x}_{n = 0}a_n(1 - x^n)\right| +\left|\displaystyle \sum^{\infty}_{n = N_x + 1}a_n x^n\right| \leqslant 2 \varepsilon \,\,\,\, \forall \varepsilon > 0 \,\,\,\, \blacksquare\]
